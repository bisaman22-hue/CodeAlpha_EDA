# Titanic Dataset Exploratory Data Analysis (EDA)

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Titanic dataset using Python.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Analysis Performed
- Data Inspection
- Missing Value Analysis
- Statistical Summary
- Survival Analysis
- Gender Distribution Analysis
- Passenger Class Distribution Analysis

## Key Findings
- Dataset contains 891 passenger records.
- Age column contains missing values.
- Cabin column contains a large number of missing values.
- More passengers died than survived.
- Passenger distribution varies across gender and class categories.
